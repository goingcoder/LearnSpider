    HTTP无状态协议是指协议对于事务处理没有记忆能力。缺少状态意味着如果后续处理需要前面的信息，则它必须重传，这样可能导致每次连接传送的数据量增大。
另一方面，在服务器不需要先前信息时它的应答就较快。                   ————百度百科
    Internet协议分为两大类。有状态协议w与无状态协议。两者最大的差别是客户与服务器之间维持联机上的不同。httpy协议即是一种无状态的协议。httpy协议采用
“连接—请求——应答——关闭连接”模式。一旦客户端的请求结束，服务器便会中断连接。不会一直与客户端保持联机的状态。当下一次请求发起时，服务器会把这个请求看成
一个新的连接，与之前的请求无关。对于交互式的web应用，保持联机状态是非常重要的。
    开始，我没有使用cookie的时候，获取到的是未登录状态的页面。在我加上cookie之后，登录成功。
